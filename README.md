**Cat Browser**

Select the Breed of cats and view some information about that particular breed.

**Technologies**

React, React Router, Vite, CSS

**Node.js Version requried**

Minimum node version v20.x

**How to clone this repo**

Clone the repository to your local repository.

    Fork the repository
    On your new forked repository click on code and copy https web URL
    Open your terminal and type in git clone and the url e.g git clone https://github.com/your-username/your-repo-name.git
    Type code . to load visual studio code.

In visual studio code click the terminal and then new terminal

When the terminal has loaded install dependencies using npm install in the terminal.

use npm run dev to start Vite development server
